R - это отношение, совокупность экземпляров. То есть некоторая таблица, где строка - экземпляр, а столбикти - атрибуты экземпляра. 

---
# Реляционные операторы

Выборка(Selection,  $\sigma$) - выбор строк по условию
Проекция(Projection, $\pi$) - Выборка столбцов
Деление(Division, ÷) - найти записи из одной таблицы, связанные со всеми записями в другой.
Объединение(union, ∪) - Объединяет строки из двух таблиц, убирая дубликаты.
Пересечение (INTERSECT, ∩) - Возвращает только те строки, которые есть в двух таблицах.
Разность (EXCEPT, −) - Возвращает строки из первой таблицы, которых нет во второй.
Произведение (декартово произведение таблиц, ×) - все возможные пары строк двух таблиц. 
Естественное соединение(join, $\infty$ - но с квадратными краями)  - автоматически объединяет таблицы по всем столбцам с одинаковыми именами.

>подвиды JOIN. 
>- **INNER JOIN** - только совпадающие записи 
>- **LEFT/RIGHT JOIN** - все из одной + совпадающие из другой 
>- **FULL JOIN** - все из обеих таблицы
