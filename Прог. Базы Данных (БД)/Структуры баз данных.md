# Ключи
**PK - Primary Key (Первичный ключ)**

- Уникальный идентификатор записи в таблице
- Не может быть NULL
- Не может повторяться
- Одна таблица = один PK (может быть составным)

**FK - Foreign Key (Внешний ключ)**

- Ссылка на PK другой таблицы
- Создает связь между таблицами
- Может повторяться и быть NULL

**UK - Unique Key (Уникальный ключ)**

- Значение должно быть уникальным (как PK)
- НО: может быть NULL
- Может быть несколько UK в одной таблице
